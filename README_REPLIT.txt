REPLIT QUICK START

1. Go to https://replit.com on your iPhone
2. Create a new Python Repl
3. Upload all files from this ZIP
4. Open the Shell tab and run:
   pip install -r requirements.txt
5. Press Run

If Replit asks for the run command, use:
python app.py

Then open the public web URL Replit gives you.
That URL will work on your iPhone and the PDF download comes from the server, not JavaScript.
